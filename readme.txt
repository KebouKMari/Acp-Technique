****************************************************** Version Anglaise ***********************************************************************


Abstract: This project consists in implementing from scratch the PCA technique. This method consists in reducing the dimensions of a set of data of very large size 
into a small size dataset where the data are quite correlated.

NB: To run the program, in your terminal, go to the directory with the .py files.	
    Then in your terminal type the command : python acp.py and it's done.
